# Изображения

Файлы в этой папке создаются командой:

```bash
python3 scripts/fetch_images.py
```

Скрипт ищет открытые материалы на Wikimedia Commons, сохраняет их локально и
обновляет `data/image_sources.json`. Если подходящий материал не найден или
скачивание не удалось, создаётся нейтральный SVG-фон с пометкой в источниках.
